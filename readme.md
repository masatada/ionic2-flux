## ionic2 flux
> ionic2 on flux 

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](http://opensource.org/licenses/MIT)

## Installation

```bash
$ npm install ionic cordova -g
$ npm install
```

## Debuging on browser
```bash
$ ionic server
```

## Production Build
```bash
$ ionic build
```

## Running Tests
```bash
$ npm test
```
